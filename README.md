# Password Strength Classifier 🔐

This project uses Machine Learning and NLP techniques to classify passwords into:

- Weak
- Medium
- Strong

## 📊 Project Workflow
1. Data Exploration
2. Data Preprocessing
3. Model Training
4. Model Evaluation

## 📂 Project Files

- 01_data_exploration.ipynb
- 02_preprocessing.ipynb
- 03_model_training.ipynb
- 04_evaluation.ipynb
- app.py

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- CatBoost
- NLP
